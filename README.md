## 📝 Descripción General
El proyecto está estructurado en una serie de laboratorios interactivos (Jupyter Notebooks) que documentan el proceso completo de Inteligencia de Negocios y Ciencia de Datos. Desde la ingesta de datos en bruto y el tratamiento de valores atípicos/nulos, hasta la consolidación de información limpia lista para el análisis y la toma de decisiones.

## 🎯 Objetivo
Aplicar técnicas de Extracción, Transformación y Carga (ETL) y Análisis Exploratorio de Datos (EDA) para identificar patrones, limpiar anomalías en los registros y transformar datos crudos en información estratégica y visualmente interpretable.

---

## 📂 Datasets Utilizados
Los datos analizados se encuentran en la carpeta `DataSet/` e incluyen:
* 📄 `dataset_sucio_practica.csv` - Utilizado para prácticas de limpieza y tratamiento de datos inconsistentes.
* 🎬 `netflix_titles.csv` - Catálogo de títulos para análisis de tendencias y agrupaciones.
* 🧪 `test.csv` - Datos de prueba para evaluación de modelos o scripts.
* 🛒 `ventas-por-factura.csv` - Registros transaccionales operativos (crudos).
* 📈 `ventas_limpias.csv` - Conjunto de datos resultante tras el proceso de normalización y limpieza.
